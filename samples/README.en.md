# alpha-visualizer Bundled Sample Data

> [日本語](./README.md) / **English**

This directory contains a self-contained sample dataset for alpha-visualizer and a deterministic regeneration script.

## Overview

`samples/sample-forge/` is a ready-to-serve forge project: launch it with `vis serve --forge-dir samples/sample-forge` to see alpha-visualizer in action. **All bundled data is fully synthetic** — no real financial instrument prices are included. The dataset is redistribution-free and commercial-use-free under the MIT license.

> Step 1 wires up the skeleton only. The actual sample dataset is generated by subsequent steps.

## Usage

```bash
uv run vis serve --forge-dir samples/sample-forge --no-open
```

## Regenerate

```bash
uv run python samples/build_samples.py
```

The script is fully deterministic (seeds are fixed) — re-running it produces byte-identical output.

## Layout (target)

```
samples/
├── README.md                          # 日本語版
├── README.en.md                       # this file
├── build_samples.py                   # deterministic regeneration script
├── _generators/                       # internal modules (do not call directly)
│   ├── synthetic_ohlcv.py             # per-symbol regime-based OHLCV synthesis
│   ├── strategy_defs.py               # 8 strategy JSON definitions (hand-copied)
│   ├── pseudo_backtest.py             # 40-run pseudo backtest generation
│   ├── pseudo_wfo.py                  # Walk-Forward Optimization pseudo results
│   ├── pseudo_grid.py                 # Grid optimization pseudo results
│   ├── compatibility_matrix.py        # strategy × symbol compatibility table
│   └── writers.py                     # SQLite / JSON / YAML writers
└── sample-forge/                      # target for `vis serve --forge-dir`
    ├── forge.yaml
    └── data/
        ├── results/backtest_results.db
        ├── strategies/*.json          # 8 files
        └── ideas/ideas.json
```

## License and Caveats

- This sample dataset is **fully synthetic** and contains no real financial instrument prices.
- Symbol names always carry the `_SYNTH` suffix (`EQUITY_SYNTH`, `INDEX_SYNTH`, `COMMODITY_SYNTH`, `FX_SYNTH`, `CRYPTO_SYNTH`). Do not confuse them with real tickers such as `AAPL` or `BTC`.
- The 8 bundled strategies are **textbook-style combinations** of technical indicators (SMA / RSI / MACD / Bollinger / ADX / Donchian). HMM-based and proprietary optimized strategies are intentionally excluded.
- For commercial backtesting engines, optimization, and live trading, please use [alpha-forge](https://github.com/ysakae/alpha-forge) (commercial license).
